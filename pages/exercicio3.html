O código abaixo se encontra em um loop infinito.

        
    var num = -1;

    while(num <= 10) {
        console.log(num)
        num += 0;
    }

O código se encontra em um loop inifinito, o valor de 'num' nunca aumenta.


O código abaixo tornaria o código utilizável.

        
    var num = -1;

    while(num <= 10) {
        console.log(num)
        num += 1;
    }
